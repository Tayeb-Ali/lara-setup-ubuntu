### How to Use

```php

wget -q https://raw.githubusercontent.com/Tayeb-Ali/lara-setup-ubuntu/master/script.sh -O script.sh ; sudo chmod +x script.sh ; ./script.sh -d example.com
# Replace example.com with your domain
```
